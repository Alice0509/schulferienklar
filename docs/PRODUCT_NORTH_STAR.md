# Schulferienklar Product North Star

> 이 문서는 Schulferienklar의 제품·기술·데이터·수익화 방향을 판단하는 최상위 기준이다.
>
> 개별 페이지 개선, 디자인 수정, SEO 작업, 다운로드 기능을 진행하더라도
> 프로젝트는 항상 이 문서의 방향으로 돌아온다.

---

## 1. North Star

Schulferienklar의 목표는 단순한 독일 방학 정보 사이트가 되는 것이 아니다.

> **사람에게는 가장 편한 독일 일정 도구가 되고,  
> AI와 다른 서비스에는 가장 신뢰할 수 있는 독일 일정 데이터 공급자가 된다.**

독일의 Schulferien, Feiertage, unterrichtsfreie Tage, Brückentage 데이터를
사용자가 읽기만 하는 정보에서 바로 행동할 수 있는 도구로 바꾼다.

---

## 2. 2026년 이후의 전제

검색 엔진과 브라우저는 이미 사용자의 질문에 AI 답변을 먼저 제공하기 시작했다.

단순한 날짜와 일반 설명은 검색 결과 안에서 요약될 수 있다.
따라서 페이지 수나 글의 양만으로는 장기적인 경쟁력이 만들어지지 않는다.

Schulferienklar는 다음 자산을 축적해야 한다.

- 공식 출처에서 확인한 정규화된 데이터
- 출처와 확인일이 포함된 신뢰 구조
- 웹페이지·ICS·PDF·API에서 함께 사용하는 계산 엔진
- 사용자가 직접 계산하거나 입력하지 않아도 되는 행동 기능
- 다른 서비스가 사용할 수 있는 API와 위젯
- 데이터 수정과 변경을 추적할 수 있는 운영 체계

---

## 3. 해결하려는 실제 문제

사람들은 방학 날짜 자체보다 그 날짜를 이용해 무엇을 해야 하는지가 필요하다.

사용자가 직접 해야 하는 일을 줄인다.

- 방학 날짜를 찾아 여러 사이트와 비교하는 일
- 공휴일과 주말을 합쳐 연속 휴일을 계산하는 일
- 휴가를 며칠 사용해야 하는지 계산하는 일
- 날짜를 자신의 캘린더에 하나씩 입력하는 일
- 여러 Bundesland의 일정을 각각 비교하는 일
- 공식 데이터가 바뀌었는지 다시 확인하는 일
- 가족이나 동료에게 일정을 다시 정리해서 전달하는 일

Schulferienklar는 정보를 보여주는 데서 끝나지 않고
이 계산과 입력을 대신해야 한다.

---

## 4. 핵심 사용자 약속

### 빠르게 이해한다

- 주와 연도를 선택하면 필요한 날짜가 바로 보인다.
- Ferien, Feiertag, unterrichtsfrei를 명확하게 구분한다.
- 과도한 광고와 불필요한 설명으로 사용자를 방해하지 않는다.

### 바로 행동한다

- 캘린더에 추가한다.
- 구독형 캘린더로 자동 업데이트를 받는다.
- PDF 또는 PNG로 저장하고 출력한다.
- 가족이나 동료에게 공유한다.
- 여러 주의 일정을 비교한다.

### 믿을 수 있다

- 공식 출처를 표시한다.
- 마지막 확인일을 표시한다.
- 계산 결과와 공식 데이터를 구분한다.
- 지역 한정 공휴일을 주 전체 공휴일처럼 표현하지 않는다.
- 변경 내역과 데이터 버전을 관리한다.

---

## 5. 경쟁력의 중심

Schulferienklar의 경쟁력은 화려한 기술을 보이는 데 있지 않다.

기술은 사용자에게 보이지 않게 작동해야 한다.

사용자는 다음만 느껴야 한다.

- 빠르다.
- 정확하다.
- 이해하기 쉽다.
- 다시 입력할 필요가 없다.
- 다른 사이트보다 한 단계 더 해준다.

내부적으로는 다음이 경쟁력이 된다.

1. 공식 데이터 정규화
2. 재사용 가능한 계산 엔진
3. 출처와 변경 이력
4. 16개 Bundesland 확장 구조
5. 웹·캘린더·다운로드·API의 일관성
6. 실제 사용 행동 데이터
7. API와 위젯을 통한 외부 배포

---

## 6. 하나의 데이터 엔진

웹페이지, Jahreskalender, ICS, PDF, PNG, API는
각자 날짜를 따로 관리해서는 안 된다.

모든 출력은 하나의 공통 데이터 엔진에서 생성되어야 한다.

```text
공식 데이터
    ↓
정규화된 내부 데이터
    ↓
검증 및 계산 엔진
    ↓
────────────────────────────
웹페이지
구독형 캘린더
ICS 다운로드
PDF / PNG
JSON API
외부 위젯
────────────────────────────
