Datenschutz

Schulferienklar ist ein statisches Informationsangebot zu Schulferien, Feiertagen und freien Zeiten in deutschen Bundesländern.

Keine Benutzerkonten

Schulferienklar bietet derzeit keine Benutzerkonten, kein Login und keine serverseitige Speicherung persönlicher Profile an.

Lokale Speicherung im Browser

Die zuletzt ausgewählten Werte für Bundesland und Jahr können lokal im Browser gespeichert werden. Diese Speicherung erfolgt über localStorage und bleibt auf deinem Gerät.

Hosting

Die Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Website können technisch notwendige Zugriffsdaten durch GitHub verarbeitet werden.

Externe Links

Schulferienklar kann auf externe Seiten verlinken, zum Beispiel auf GitHub. Für Inhalte und Datenschutz dieser externen Seiten sind die jeweiligen Anbieter verantwortlich.
